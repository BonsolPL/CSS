1. Składnia klasy w CSS

    2. selektor.klasa { cecha: wartość }

  2. Opis klasy, cechy i wartości

    3. Klasa to atrybut class="..." przypisany w znaczniku HTML.
    "cecha" i "wartość" definiują wygląd elementu w CSS.

  3. Przykładowe selektory

    4. p => paragraf

    h1 => nagłówek

    td => komórka tabeli

  4. Uwagi o nazwach klas

    5. Nazwy klas to pojedyncze wyrazy, bez:
    a) spacji
    b) kropek
    c) przecinków
    d) dwukropków
    e) nawiasów
    Zawierają:
    a) litery
    b) cyfry
    c) myślniki
    d) podkreślenia
    Nie mogą zaczynać się od cyfr ani znaków specjalnych.

  5. Zastosowanie klas

    6. Używamy klas dla wielu elementów z tym samym wyglądem, niezależnie od ich typu HTML.

  6. Użycie klas przy ograniczeniach selektorów

    7. Kiedy nie chcemy, by wszystkie elementy tego samego typu wyglądały jednakowo.

  7. Składnia wielu klas

    8. <selektor class="klasa1 klasa2 klasa3">...</selektor>

  8. Podzbiory klas

    9. selektor.klasa1.klasa2 { cecha: wartość }

    Element musi mieć przypisane wszystkie klasy jednocześnie, by reguła zadziałała.

  9. Kiedy stosować klasy

    10. Gdy trzeba przypisać kilka klas do jednego elementu.

  10. Planowanie klas

    11. Gdy na stronie wiele elementów wymaga podobnego stylu, a selektory typu są niewystarczające.

  11. Komentarze przy klasach

    12. Komentarze pomagają pamiętać, do czego służy dana klasa po dłuższym czasie.

  12. Definicja i wywołanie klasy

    13. p.testowa-klasa { color: red }

    <p class="testowa-klasa">To jest tekst z klasą.</p>

  13. Klasa dla jednego selektora

    14. h2.tylko-dla-naglowka { color: green; font-size: 16pt }

  14. Klasa uniwersalna i znak "*"

    15. .globalna { font-style: italic; color: navy; }

    <div class="globalna">Ten element korzysta z klasy uniwersalnej.</div>
