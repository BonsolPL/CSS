1. Opis znacznika DIV

Element blokowy. DIV jest kontenerem do grupowania elementów HTML.

<div align="wartość">Treść</div>

Zastosowanie: grupowanie elementów w sekcje, stosowanie CSS i JavaScript.

Atrybuty: id, class, style, title, align.

ALIGN: Wyrównuje tekst wewnątrz DIV: left, center, right, justify.

2. Opis znacznika SPAN

Element liniowy. SPAN jest używany do stylizacji części tekstu.

<span>Treść</span>

Zastosowanie: stylizacja części tekstu, dodawanie efektów JavaScript.

3. Różnica między znacznikami DIV i SPAN

DIV : Element blokowy – zajmuje całą szerokość.

SPAN : Element liniowy – obejmuje tylko swoją treść.

4. Elementy HTML

Elementy liniowe: span, a, b, i, img, input, label.

Elementy blokowe: div, p, h1-h6, ul, ol, table.

Elementy blokowo-liniowe: button, textarea.
