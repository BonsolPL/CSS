• :first-line

    Pozwala sformatować tylko pierwszą linię tekstu w elemencie.

    div { color: teal; font-size: 14pt; } div:first-line { color: darkred; font-size: 22pt; text-transform: uppercase; }

  • ::before i ::after

    Służą do dodawania zawartości przed i po elemencie HTML za pomocą CSS.

    div::before { content: "<<< POCZĄTEK >>>"; font-size: 14pt; font-style: oblique; color: green; } div::after { content: "<<< KONIEC >>>"; font-size: 14pt; font-style: oblique; color: green; }

  • :first-letter

    Formatowanie tylko pierwszej litery bloku tekstu.

    div { color: navy; font-size: 28pt; } div:first-letter { color: orange; font-size: 90pt; font-weight: bold; }
