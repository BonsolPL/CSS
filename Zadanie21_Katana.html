Zadanie 21 - Identyfikatory ID

Pojęcie Znaczenie
Co znaczy skrót ID? ID oznacza identyfikator.
Jaki jest cel stosowania ID? Umożliwienie unikalnego identyfikowania elementów na stronie.
Jakie jest znaczenie znaku #? Znak # jest używany przed nazwą identyfikatora w CSS.
Jak następuje wywołanie ID? W HTML: <element id="nazwazdefiniowanawCSS">
W CSS: #nazwazdefiniowanawCSS { stylizacja }
Uwagi na temat stosowania ID 1. Unikalność: Element oznaczony ID musi być unikatowy na jednej stronie.
2. Raz użyty: Identyfikator może wystąpić tylko raz w całym dokumencie.
3. Klasa vs ID: Klasy można używać wielokrotnie, ale ID musi być unikatowe.
