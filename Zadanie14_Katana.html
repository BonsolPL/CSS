Sposoby Definiowania Kolorów

Kolory w CSS można definiować na różne sposoby. Poniższa tabela przedstawia najczęściej używane metody wraz z przykładami.

Metoda Opis Przykład
Kolory nazwane Użycie nazw kolorów zdefiniowanych w standardzie CSS. color: red;
RGB Definiowanie koloru za pomocą składowych czerwonej, zielonej i niebieskiej (Red, Green, Blue). color: rgb(255, 0, 0);
RGBA RGB z dodatkiem kanału alfa określającego przezroczystość. color: rgba(255, 0, 0, 0.5);
HEX Definiowanie koloru za pomocą szesnastkowego kodu koloru. color: #ff0000;
HSL Definiowanie koloru za pomocą składowych Hue (odcień), Saturation (nasycenie) i Lightness (jasność). color: hsl(0, 100%, 50%);
HSLA HSL z dodatkiem kanału alfa określającego przezroczystość. color: hsla(0, 100%, 50%, 0.5);
